# DLM — Dynamic Layered Media

**Live:** [dlm-tau.vercel.app](https://dlm-tau.vercel.app)

A new audiovisual format. Place sounds anywhere on your photo — let people discover them by touch.

## What is a .dlm file?

A `.dlm` file is a single portable file that carries a photo + sound pins + audio data. Open it on any device, no app needed.

## Features

- **Editor** — upload a photo, place draggable sound pins, record live or upload audio, export as `.dlm`
- **Viewer** — open any `.dlm` file and tap the glowing points to hear the sounds
- Bilingual EN / PT
- Fully responsive (mobile + desktop)
- No account, no install, runs entirely in the browser
